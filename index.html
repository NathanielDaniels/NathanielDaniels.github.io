About Me

Hey, my name is Nathaniel and I'm a driven Self-taught Web Developer from Chicago.


I have mainly focused on Front-End Web Development. What really peaked my interest was the ability to start with an empty slate and build anything you can imagine. Once I got going, I realized that I'm a huge fan of designing with CSS. Once I learned positions, I though I could do it all. Well, that was before I found out about Flexbox and Grid layouts... my whole World Changed. Things became much clearer from that point forward, and my code, much cleaner.

Next, came the crazy adventure through javascript. Lots of mind blowing moments, especially while learning animations with JQuery. I could tinker with animations all day. Though, Thanks to a few clients, I started into learning back-end to set up some forms. I had to quickly learn how to build a database to store data, then link it to my project. Now I have a working form at the bottom of this website that I set up myself.



Font-End: I know HTML, CSS/SASS. Javascript, ES6 and JQuery + Node.js (mostly NPM). GIT/Github. I'm very active with the Terminal. I also know how to build HTML Emails. Currently, I'm learning React.


Back-End: I have worked with PHP and MySQL.


I am currently interested in both Front-End Web Development and HTML Emails.



If i'm not coding up something new, you'll find me searching the internet for new ideas, fresh tutorials and tips and tricks. Soon I'll be writing my own blogs/articles for everyone to learn that next big thing, or just my perspective. Even though I haven't been in this industry for very long, I strive to become a big name in web development.
